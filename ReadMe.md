# Interactive User Card Generator

This project focuses on mastering DOM manipulation and event handling by building an interactive user card generator. It demonstrates dynamic creation and removal of elements, real-time UI updates through event listeners, role promotion logic, search-based filtering, and smooth UI animations with Tailwind CSS.

## Features

- Dynamic Card Creation
User cards are generated dynamically from a JavaScript array using createElement and appendChild.
- Promote & Remove Buttons
Users can be promoted through roles (Member → Senior Member → Veteran) or removed entirely, with smooth transition animations.
- Event Delegation
Efficient handling of multiple user card interactions using a single event listener on the container.
- Search Filtering
Real-time filtering of user cards as you type, with a graceful “no results” message when no matches are found.
- Tailwind CSS Animations & Styling
Responsive grid layout, hover effects, rounded cards, and smooth transition animations for promotion and removal actions.
- Responsive Design
Cards adapt to multiple screen sizes using Tailwind’s grid system and utility classes.
- Semantic HTML Structure
Organized markup for readability and maintainability.

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)

## Purpose

This project reinforces key frontend engineering concepts:

- Manipulating the DOM dynamically and efficiently
- Handling user interactions with event listeners and delegation
- Implementing real-time search and filtering in the UI
- Using CSS frameworks (Tailwind) for responsive and interactive designs
- Structuring JavaScript code for readability and maintainability
